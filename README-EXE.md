# WavEDA 仿真软件助手（Windows 免安装版）

1. 完整解压 ZIP，不要在压缩包预览窗口中直接运行。
2. 双击 `WavEDA_Assistant.exe`。
3. 在“设置 > API 设置”中填写自己的 API Key。
4. 如需调用本机 WavEDA 图片，可在“设置 > WavEDA 路径”填写自己的安装目录，例如 `D:\Program Files\WavEDA`。
5. 保存设置后重启软件生效。

不填写 API Key 时仍可使用本地知识库检索。请保留 EXE、`_internal`、`knowledge_base`、`wavEDA_docs`、`data` 和 `config` 的相对位置，不要只单独移动 EXE。

建议把软件放在桌面、文档或其他有写入权限的位置，不要放进 `C:\Program Files`。API Key、WavEDA 路径和聊天记录只保存在本机。
