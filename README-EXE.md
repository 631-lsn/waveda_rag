# WavEDA 仿真软件助手（Windows 免安装版）

1. 完整解压 ZIP，不要在压缩包预览窗口中直接运行。
2. 双击 `WavEDA_Assistant.exe`。
3. 首次启动时可以选择自动创建桌面快捷方式；之后也可在设置中重新创建。
4. 发布版已经预设 API 服务，可以直接提问；如需更换，可在“设置 > API 设置”中修改。
5. 如需调用本机 WavEDA 图片，可在“设置 > WavEDA 路径”填写自己的安装目录，例如 `D:\Program Files\WavEDA`。
6. 修改设置后重启软件生效。

请保留 EXE、`_internal`、`knowledge_base`、`wavEDA_docs`、`data` 和 `config` 的相对位置，不要只单独移动 EXE。

建议把软件放在桌面、文档或其他有写入权限的位置，不要放进 `C:\Program Files`。API Key、WavEDA 路径和聊天记录只保存在本机。
