WavEDA

主页

主页的功能介绍，如下图示意，左侧的矩形框内包含着WavEDA文件的快捷键，每个快捷键表示菜单中的一个可用命令。初始功能（打开/导入，关闭，保存，另存为）提供了用于添加和打开项目，选择求解器，查看案例或者寻求帮助。右侧的矩形框内包含历史文件。单击文件名后，中间的矩形框内是图像显示窗口会显示文件对应模型。 点击后可以快速进入。底部的矩形框包含着WavEDA文件的仿真模块。

打开

点击打开，弹出文件夹后，你可以选择任意的“tsp”文件打开。

关闭

点击关闭，弹出确认窗口后，再次点击确认，可以关闭WavEDA。如果你不小心误触关闭弹出确认窗口后，点击关闭，则可以回到主页。

保存

点击保存，弹出文件夹后，你可以选择任意的位置保存WavEDA文件。

另存为

点击保存，弹出文件夹后，你可以选择任意的位置另保存WavEDA文件，注意修改文件名称。

选项

点击选项，页面将切换到WavEDA的设置页面，在这里你可以配置WavEDA的许可证书，点击浏览，选择可以的许可证。

案例

点击案例，案例库内有多种仿真模型可供参考，目前案例库内包含多种端口的模型，双击图片后，可以看到关于这个模型的介绍，包括模型描述、仿真设置和后处理结果。双击名称可直接打开案例文件。

帮助

点击帮助，可打开WavEDA的帮助文档。
