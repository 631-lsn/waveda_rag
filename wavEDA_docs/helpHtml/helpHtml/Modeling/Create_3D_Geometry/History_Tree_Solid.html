历史树-体

双击编辑后的体，在体的属性窗口中记录体的所有操作步骤，即体的历史树。 WavEDA的体-历史树支持模型编辑和历史操作撤回等操作，下面将逐一介绍。

布尔操作

在历史树中可直接对该模型与列出的几何形状进行布尔运算。

操作步骤：

  1. 选中在相加、相减或相交。
  2. 点击另一种模型类型进行创建。
  3. 创建过程中选择预览，确定另一个模型的位置和大小。
  4. 点击完成，查看效果。

以一个正方体加上一个小球为例：

  1. 双击正方体，打开历史树。
  2. 选中相加布尔操作，点击圆，填入尺寸。
  3. 点击预览查看效果，点击完成。
  4. 此时即完成了在历史树中完成一次布尔操作的操作，且该此操作加在了历史树中。

撤销操作

选中之前对模型的编辑操作，可进行撤销，还原等操作。图中依次为删除，解除关联以及恢复操作。

  • 删除：选中操作，点击删除，即可删除操作。
  • 解除关联：解除该操作时，两个或多个物体之间的关联。
  • 恢复：将原始物体恢复。
