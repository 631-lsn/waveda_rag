# WavEDA 快速入门

新手做一次电磁仿真，可以按下面顺序检查：几何模型 -> 材料 -> 端口 -> 边界条件 -> 计算区域 -> 求解配置 -> 网格 -> 运行仿真 -> 结果后处理。

## 1. 几何与单位

导入 STP/STEP、IGES、DXF、Gerber 等文件后，先确认单位是否正确。不同软件的默认单位可能不同，WavEDA 中可在设计工程树下的“单位”里选择单位，并根据实际情况勾选“缩放模型”或“保持值不变”。单位错会导致尺寸、频率响应、端口大小和网格密度全部偏离。

## 2. 材料设置

在左侧工程树中找到体或立体块，找到希望赋予或修改材料的对象后右击，进入“属性”。弹出的选项中通常会有编辑、编辑材料、修改材料、修改颜色、修改透明度、修改网格尺寸、修改 DDM 网格尺寸等操作。

一般需要给每个参与仿真的立体都赋予材料，否则求解时容易报错或结果异常。做跨软件对比时要特别注意：WavEDA 的材料参数名和其他软件不一定完全一致，折射率相关模块仍在开发时，应优先用电磁模块中的相对介电常数来替代配置。

## 3. 端口设置

端口方向在预览时应该朝外。由于端口算法的方向约定，预览箭头朝外时，实际激励方向通常就是向结构内部传播。设置端口后一定要预览，确认端口面、方向、尺寸和模式数都正确。

端口模式数会影响 WavEDA 计算哪些模式的 S 参数。编辑激励时把某个端口模式的幅度设为 1、其他设为 0，主要影响快照 snapshot 的激励显示，不会改变 S 参数本身；WavEDA 会按端口设置中的模式数计算对应模式结果。

## 4. 边界条件

边界条件可以在电磁工程树中找到。常见类型包括周期性边界条件 PBC、Symmetry、辐射边界条件 ABC、完美匹配层 PML、理想电导体 PEC、理想磁导体 PMC、阻抗边界条件 1BC、Spectral Integral Boundary SIB、表面阻抗边界条件 SIBC、表面电流边界条件 SCBC，以及默认 ABC。

设置边界时可以选择区域，例如 +X、-X、+Y、-Y、+Z、-Z，也可以由用户自定义面。完成前建议预览边界覆盖范围，确认没有漏选、错选或端口与边界冲突。边界设置错误或不完整会明显影响仿真结果。

## 5. 计算区域

计算区域可以选择“使用盒子包围所有物体-无间距”来降低仿真内存，也可以选择“仅包含所有的 3D 物体”达到类似效果。根据模型需要，也可以在间距下拉项中选择自动间距、无间距、User defined gap (absolute)、User defined gap (percentage) 或自定义空气盒子，并在 +/-X、+/-Y、+/-Z 方向设置对应参数。

## 6. 求解配置

求解配置一般在设计工程树下的“配置”中设置。可选择时域或频域；频域频率设置通常包括单频、步长、数值、对数型和多频点。

扫频算法要和任务目标匹配：离散扫频最直接但通常最慢；插值扫频更快但不支持快照；快速扫频适合快速看趋势，但不支持自适应网格。数值设置新手一般使用默认值，结果异常时再有记录地调整。

## 7. 运行与后处理

运行前检查材料、端口、边界、计算区域、求解器、网格是否完整。仿真完成后，可在电磁结果中查看端口结果、S 参数曲线、场图和 snapshot。若需要导出 S 参数，优先参考 FAQ 中的导出路径说明。
