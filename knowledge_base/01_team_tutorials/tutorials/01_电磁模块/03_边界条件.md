# 边界条件

边界条件决定计算区域外侧如何近似真实物理环境。边界设置错误或不完整，会显著影响 S 参数、场分布和辐射结果。

## 设置入口与选择方式

边界通常可以在电磁工程树中找到。设置时可以选择区域，例如 +X、-X、+Y、-Y、+Z、-Z，也可以选择用户自定义面。完成前建议使用预览确认覆盖范围。

## 常见边界类型

- Default ABC：默认吸收或辐射类边界。
- PBC：周期性边界条件，适合周期结构。
- Symmetry：对称边界，用于利用几何或场分布对称性减少计算量。
- ABC：辐射边界条件，用于近似开放空间。
- PML：完美匹配层，用于吸收向外传播的波。
- PEC：理想电导体边界。
- PMC：理想磁导体边界。
- 1BC：阻抗边界条件。
- SIB：Spectral Integral Boundary。
- SIBC：表面阻抗边界条件。
- SCBC：表面电流边界条件。

## 新手注意点

边界可按区域统一设置，也可对局部面单独设置。不要只依赖默认边界而不检查；开放空间、周期结构、金属壁、对称面和端口附近的物理含义不同。

端口附近的边界尤其要谨慎。端口面、PML、PEC/PMC 等如果重叠或距离不合理，可能导致端口模式异常、S 参数异常或求解失败。

## 检查清单

- 六个方向或自定义面是否都按预期设置。
- 边界预览是否有漏选、错选。
- 是否需要 PBC、Symmetry、PML 或 PEC/PMC，而不是默认 ABC。
- 与其他软件对比时，边界类型、距离和计算区域是否一致。
