# WavEDA 错误排查指南

遇到错误时，建议按“单位 -> 几何 -> 材料 -> 端口 -> 边界 -> 计算区域 -> 网格 -> 求解器 -> license”的顺序排查。不要只盯着最后一条报错，前面的 warning 往往已经提示了问题来源。

## 材料相关错误

目前折射率模块仍在开发或不稳定时，材料设置建议使用电磁模块中的相对介电常数来替代。若材料在其他软件里以折射率 n 给出，低损耗近似下可以先用相对介电常数 εr = n^2 作为参考，但做严肃对比时需要确认损耗、频散、磁导率和单位定义是否一致。

WavEDA 的材料参数和其他软件不一定一一对应。跨软件对比时必须统一材料定义，尤其是相对介电常数、损耗角正切、电导率、金属模型和频率依赖参数。

常见现象：未给所有立体赋材料、材料参数为空、材料单位或含义填错，都会导致求解失败、结果异常或和其他软件差异很大。建议在左侧工程树逐个检查体对象，右键“属性”查看并修改材料。

## 网格划分失败或长时间卡住

| 现象 | 常见原因 | 处理建议 |
| --- | --- | --- |
| 小模型长时间卡在网格某一步 | 几何错误、单位错误、材料缺失、局部网格过细 | 不建议无限等待，先检查导入单位、几何完整性和材料赋值。 |
| 网格数量异常大 | 计算区域过大、空气盒子间距过大、特征尺寸过小 | 适当使用“仅包含所有 3D 物体”或“使用盒子包围所有物体-无间距”。 |
| 自适应网格结果不稳定 | θ 参数或误差目标过严，局部场变化剧烈 | θ 参数一般保持默认；必要时记录配置后再调整。 |

## 端口与激励问题

端口预览方向应朝外，实际激励方向通常向结构内部传播。若端口方向、端口面或端口大小设置错误，S 参数和场图都会异常。

端口模式数会影响 S 参数计算的模式范围；在“编辑激励”中改变某个模式的激励幅度，主要影响 snapshot，不改变已经按模式数计算出的 S 参数。默认激励源一般是第一个端口或第一个模式，查看快照前需要确认激励源。

## 边界与计算区域问题

边界条件设置错误或不完整会显著影响结果。边界可按区域选择 +/-X、+/-Y、+/-Z，也可自定义面。设置完后一定要预览，确认没有漏选、错选，也没有和端口发生不合理冲突。

计算区域过大可能导致内存和时间暴涨，过小则可能截断场分布。新手可优先尝试“使用盒子包围所有物体-无间距”或“仅包含所有的 3D 物体”，再根据辐射、端口和开放区域需求调整空气盒子。

## 求解器与扫频问题

离散扫频会逐点求解，通常最慢但最直接。插值扫频通过矢量拟合或有理插值估计频点，不支持快照；若插值点数过多或误差难以满足，可能说明模型没有收敛。快速扫频适合看趋势，但不支持自适应网格。

如果结果曲线出现异常尖峰、跳变或和其他软件差异很大，优先用少量关键频点的离散扫频做交叉验证。

## License 与闪退

`Failure in license request...`、`other copies of WavEDA are running simulation...` 等 license 报错通常需要联系管理员解决。可先通过 `Win + R` 输入 `services.msc` 检查 license 服务状态，但不要随意改服务配置。

闪退通常是程序自我保护机制。若某一步能稳定复现闪退，请保存工程、截图和复现步骤，提交到禅道网站报 bug。
