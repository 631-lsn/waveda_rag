# 选项

- 原始相对路径: `File/Options.html`
- 知识模块: `文件与主页`

## 正文抽取
## 选项

点击选项，页面将切换到WavEDA的设置页面，在这里你可以配置WavEDA的许可证书，调整WavEDA显示相关的配置。

> 图示要点：`Options_1.png` 展示“选项”相关的操作界面或示例，后续审查通过后再补图。

### 许可证

点击许可证，页面将切换到WavEDA的配置许可证的页面。在安装包内软件会配置一个Localhost，或者用户手动填写使用电脑的IP的地址。对于路径，点击浏览后出现弹窗要在WavEDA安装包内找到对应的exe文件，如下图，打开后运行软件将载入WavEDA内。

> 图示要点：`Options_3.png` 展示“许可证”相关的操作界面或示例，后续审查通过后再补图。

点击许可证文件后出现弹窗需要在WavEDA安装包内找到对应的lic文件，打开后许可证将载入WavEDA内。

### 配置

点击许可证，页面将切换到WavEDA的配置的页面。从上至下分别是显示全局坐标系、显示局部坐标系、显示背景网格、显示侧边栏信息、显示比例尺、显示计算区域以及设置WavEDA自动保存的时间。 下方模块，可以切换WavEDA的语言模式和仿真核数，核数越多占用电脑的内存越大，仿真速度越快。软件内默认为24核，具体的核数可通过电脑自身的硬件进行配置。

> 图示要点：`Options_4.png` 展示“配置”相关的操作界面或示例，后续审查通过后再补图。
