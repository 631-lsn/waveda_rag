# WavEDA 介绍

WavEDA 是面向电磁场仿真的工具，核心能力包括混合有限元算法、区域分解、自适应网格迭代和模型降阶等。它主要用于处理大规模、多尺度、高密度互连、高频电磁效应等复杂仿真问题。

## 核心能力

- 精确 3D 建模与分析。
- 支持复杂边界条件，例如 PML、阻抗边界、周期边界和开放边界。
- 支持多种激励类型，例如点源、平面波、集总端口和波端口。
- 支持复杂材料建模，例如高损耗电介质色散、磁性色散和各向异性材料。
- 支持后处理查看 S 参数、远场结果和快照 snapshot。
- 面向 S 参数提取、信号完整性、电源完整性、电磁兼容和射频微波仿真等场景。

## 新手理解

如果只想快速上手，可以先把 WavEDA 理解为“用于建模、设置材料/端口/边界、求解电磁响应、查看 S 参数和场图的仿真平台”。高级算法和多物理场能力可以后续再逐步学习。
