## 导入自适应网格

导入自适应网格功能使用户无需重复进行网格剖分，方便查看不同网格剖分下的仿真结果。选中网格迭代次数并点击“运行”则使用所选中网格直接开始仿真；点击“导入”则将此次网格导入后并关闭此界面，点击主窗口中的显示网格可以查看此次网格的剖分情况，或设置其他参数后进行仿真。
