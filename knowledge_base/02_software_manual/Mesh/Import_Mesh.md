## 导入网格

点击导入网格，可以将参数化扫描不同参数下的网格剖分情况导入主界面，便于在显示网格下查看不同参数的网格剖分情况，或设置其他参数后进行仿真。如下图所示是对模型中Si材料的电导率进行参数化扫描仿真，Si的电导率分别为0.02 S/m、10 S/m和100 S/m时剖分的网格，在此界面可将其中一个网格划分结果导入使用。
