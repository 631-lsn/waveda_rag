# WavEDA 仿真助手 — 完整使用手册

---

## 一、软件简介

WavEDA 仿真助手是一个**本地 RAG 知识库问答工具**，专为 WavEDA 多物理场仿真软件用户设计。你可以用自然语言提问（如"波端口怎么设置？"），AI 会自动从帮助文档、团队教程和错误案例中检索相关内容，给出简洁可操作的回答，并标注来源。

**无需联网即可使用本地检索**，如果需要大模型生成回答，可以配置 DeepSeek / Kimi / 千问 / OpenAI 等 API Key。

---

## 二、如何启动软件

软件提供两种启动方式：**开发版（.bat）** 和 **免安装版（.exe）**。

### 方式 A：免安装版（.exe）— 推荐普通用户

1. 从 Release 页面下载 `WavEDA_Assistant_Windows_YYYYMMDD.zip`
2. 解压到任意位置（桌面、文档均可，**不要放到 C:\Program Files**）
3. 双击 `WavEDA_Assistant.exe`
4. 首次启动会询问"是否在桌面创建快捷方式"，建议点"是"

优点：不需要装 Python，解压即用
缺点：文件较大（~200MB），更新需重新下载

### 方式 B：开发版（.bat）— 推荐开发者

1. 安装 Python 3.11+（勾选 "Add Python to PATH"）
2. Clone 项目仓库：`git clone https://github.com/631-lsn/waveda_rag.git`
3. 双击 `setup_env.bat`（只需首次，自动装环境+依赖+建索引）
4. 以后双击 `start.bat` 启动

优点：可以改代码、`git pull` 快速更新
缺点：需要装 Python

## 四、基础使用

### 1. 配置 API Key

1. 点击右侧 **"设置"** 按钮
2. 在 **API 设置** 标签页选择模型（DeepSeek / Kimi / 千问 / 百炼 / OpenAI）
3. 填入你的 API Key
4. 点击确定 → 重启软件

> 不配 API Key 也可以使用，只是回答由本地知识库直接生成（非 LLM 生成）。

### 2. 提问

在底部输入框输入问题，回车或点 ↑ 按钮发送。例如：

- "波端口怎么设置？"
- "PML 和吸收边界有什么区别？"
- "如何导出 S 参数？"
- "网格划分失败怎么排查？"

也可以点快捷问题按钮一键提问。

### 3. 查看来源

每条回答下方会显示来源证据——点击可查看原始文档。

### 4. 收藏与纠错

每条回答右上角有三个按钮：

| 按钮 | 功能 |
|------|------|
| **复制** | 复制回答文本 |
| **收藏** | 收藏到收藏夹，可在侧边栏查看 |
| **纠错** | 指出回答错误，AI 修正后存入知识库 FAQ |

### 5. 搜索收藏夹

点击右侧 **“收藏夹”** 打开收藏列表，在顶部搜索框输入问题关键词即可过滤收藏内容。

- 搜索范围仅为收藏项的 **问题（Q）文本**，不会搜索回答正文
- 支持中英文大小写差异和多余空格
- 例如收藏的问题是“如何导出 S 参数？”，可以搜索“导出 S 参数”
- 如果只在回答正文中出现某个词，而问题本身没有该词，不会命中

## 六、多会话功能

- 左侧对话列表显示所有历史对话
- 点击 **"+"** 新建对话
- 点击对话名称切换对话，**历史消息完整保留**

- 首次对话后 AI 自动生成 8 字以内的标题
- 启动时自动清理空白会话

## 七、知识库管理

点击 **"设置"** → **"知识库"** 标签页：

### 浏览与编辑
- 左侧树形结构展示知识库分类（团队教程/软件手册/案例/排错/参考/理论）
- 点击文件可在右侧编辑 Markdown 内容
- **右键文件**可删除

### 导入文档
1. 点击"选择文件" → 选 PDF / PPT / MD 文件
2. 选择目标分类和子目录（可选 AI 自动推荐）
3. 设置重要度（1-5）
4. **务必填写备注**（帮助 AI 准确分类）
5. 点击"确认导入"
6. AI 分析建议 → 确认 → 文件写入知识库 → 自动重建索引

### 手动重建索引

右侧 **“资料与状态”** 面板中，四张状态卡片下方有 **“重建索引”** 按钮。

适合以下情况使用：

- 手动编辑或替换了知识库文件
- 从其他电脑复制了知识库文件
- 状态卡片中的知识块数量与文件实际数量不一致
- 检索结果明显没有包含刚更新的问题

点击后程序会在后台重新扫描知识库、生成索引并重新加载检索管线。重建过程中按钮会暂时不可用，完成后状态卡片会更新。

> 程序启动时会检查知识库源文件变化；通常不需要每次重启都手动重建。只有检测不到变化或索引异常时，才需要点击此按钮。

### 检索范围说明

知识库问答优先匹配文档标题或 FAQ 中 `Q:` 后的问题标题，不会因为正文里偶然出现一个关键词就直接命中。命中后，对应分块正文仍会作为回答依据，因此回答可以包含完整操作步骤。

## 八、临时文件上传

聊天框左边的 **"+"** 按钮可以临时上传文件用于当前提问：

| 文件类型 | 说明 |
|---------|------|
| PDF / PPT / DOCX / MD | 提取文本内容附加到提问中 |
| 图片 | 需要多模态模型（GPT-4o / Claude 等）支持 |

上传后输入框上方显示附件标签，点 × 可移除。问完即弃，不写入知识库。

## 十二、常见问题

**Q: 启动后聊天区空白？**
A: 等待 2-3 秒，知识库正在加载。如果一直空白，检查 `data/index/` 目录是否存在。

**Q: API Key 配了但回答没变化？**
A: 设置完 API Key 后需要重启软件。

**Q: 导入 PDF 失败？**
A: 确保 `setup_env.bat` 已成功运行，`pypdf` 依赖已安装。

**Q: 切换语言/主题后没生效？**
A: 需要重启软件。

**Q: 检索不到刚导入的文档？**
A: 先确认文件已写入知识库，再点击右侧“资料与状态”面板中的“重建索引”按钮。重建完成后重新提问；如果仍无结果，请检查问题关键词是否出现在文档标题或 FAQ 的问题标题中。

**Q: 我已经更新了程序，但界面或搜索行为没有变化？**
A: 完全关闭所有 WavEDA 窗口后重新启动。旧进程会继续使用启动时加载的旧代码，仅关闭一个窗口可能仍有后台进程未退出。

**Q: 为什么正文里有关键词，但没有被检索到？**
A: 当前检索策略只把标题或 FAQ 问题标题作为匹配入口，正文只在命中后用于生成回答。这是为了减少正文泛化词造成的误命中。
